# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - May 11th 2026

### Added
- `Spark` typed EventEmitter3 wrapper
- History and replay support via `RingBuffer`
- Middleware support (sync and async)
- Namespace support via `createNamespace`
- Logger integration via `SparkOptions.logger`
- `emitAsync` for async middleware chains
- Full TypeScript inference for event maps
